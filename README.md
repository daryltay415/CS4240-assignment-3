# CS4240 assignment 3

Move Mode: Allow repositioning and removal of placed furniture. 
How to use: 
To reposition - clicking the screen in this mode will select the closest furniture to the indicator. Clicking again will place the selected furniture at the indicator position.
To remove - click to select furniture. then click the bin icon in the lower right side to remove.

Place mode:
How to use:
Selecting furniture - click "furniture catalog" button to explore different furniture. once selected, the furniture will appear on the indicator if able to be placed down.
Colliding furniture - If the indicator is red, it means the selected furniture is unable to be placed due to another piece of furniture colliding.
